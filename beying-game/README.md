# 背影 - 交互式文学课件

基于朱自清经典散文《背影》制作的交互式叙事网页游戏，通过游戏化的方式帮助学习者深入理解这篇经典文学作品。

## 项目简介

本游戏将《背影》拆分为7个关键场景，每个场景配有AI生成的民国风格插图，并设计了基于原文的选择题。玩家需要根据对原文的理解做出正确选择，游戏会即时提供详细的反馈，帮助深化对作品的理解。

## 游戏特色

### 1. 场景设计（7个关键场景）
- **场景1：家中变故** - 祖母去世，父亲失业
- **场景2：南京相见** - 父亲坚持送行
- **场景3：车站送别** - 浦口车站的准备
- **场景4：叮嘱茶房** - 父亲的细致关怀
- **场景5：买橘子的背影** - 全文核心场景
- **场景6：父子告别** - 父亲离去的背影
- **场景7：来信惦念** - 收到父亲来信

### 2. 图片风格
- 民国时期写实油画风格
- 怀旧暖色调（褐色、米黄、暖灰）
- 统一的视觉语言，营造时代氛围
- 所有图片由AI生成（Doubao SeeDream 4.0）

### 3. 交互设计
- 每个场景3个选项（1正确 + 2错误）
- 正确选项严格基于原文情节
- 错误选项看似合理但偏离原文或人物性格
- 详细的反馈系统，引用原文并解释对错原因

### 4. UI风格
- 民国复古风格设计
- 楷体字体营造书卷气息
- 暖棕色主色调，配合怀旧氛围
- 响应式布局，支持移动端

## 使用方法

### 快速开始
1. 用浏览器打开 `index.html` 文件
2. 阅读场景文本，观察场景插图
3. 根据对《背影》原文的理解选择答案
4. 查看反馈，了解选择的对错原因
5. 继续下一场景，直到完成所有7个场景

### 计分规则
- 每个场景回答正确得1分
- 总分7分，游戏结束后显示得分和评价
- 可以随时重新开始游戏

### 评分标准
- **100%**：完美理解《背影》的情感和细节
- **80%+**：优秀，很好地把握了作品的主旨
- **60%+**：良好，理解了主要内容
- **60%以下**：需要加强，建议重读原文

## 技术实现

- **纯HTML/CSS/JavaScript**：无需服务器，单页面应用
- **响应式设计**：支持桌面端和移动端
- **CSS动画**：平滑的场景切换和反馈效果
- **民国风格UI**：自定义样式，营造时代感
